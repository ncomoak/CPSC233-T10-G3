To run the file on eclipes:
1. Create a new Java Project, uncheck "default location" and set the directory to the location of the file.
"You might have to hover over the maze class and let Eclipse Change it's name"
2. Right click the project and click properties '
3. Select Java build path "On the left"
4. Click add class folder.
5. find the res folder and add it.
6. Go to the laucher class and run it.
